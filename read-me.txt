Locally Run with Docker Compose: 
1. Start docker daemon (docker desktop or similar) 
2. Go to the directory of docker-compose.yml file from terminal;
3. run command: docker-compose up
4. open browser at "localhost:4200/admin" 
5. use default credential to login 
    username: admin@shopizer.com
    password: password
